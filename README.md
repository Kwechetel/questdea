# QuestDea

QuestDea is an interactive learning platform that combines knowledge discovery with engaging visual elements. The platform features an interactive particle system that represents different categories of learning and personal development.

## Features

- Interactive particle background with educational facts
- Four main knowledge categories:
  - Brain Power
  - Learning
  - Wellness
  - Productivity
- Responsive design for all devices
- Beautiful UI with gradient effects and animations
- Interactive knowledge discovery through particle interaction

## Technologies Used

- React
- TypeScript
- Material-UI
- Canvas API for particle effects

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/questdea.git
```

2. Install dependencies:

```bash
cd questdea
npm install
```

3. Start the development server:

```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
